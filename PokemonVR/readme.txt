El participante entregará una aplicación que muestre la información completa de un Pokemon
Los puntos a evaluar son los siguientes:
- Tener configurado un Launch Screen para la aplicación
- Tener configurados los iconos de la aplicación
- Soportar modo obscuro y light (agregar por lo menos set color que se ocupen en la intefaz)
- Al seleccionar un pokemon se debe lanzar un nuevo view controller que muestre los siguiente datos del pokemon: nombre, imagen, movimiento y habilidad.
- En el view controller de detalle agregar un botón que "cierre" el view controller modal.
